Neovim에서 render-markdown.nvim 플러그인을 사용하면 마크다운 문서를 보기 좋게 그려준다.

## LaTex 구문 강조

### 사전 준비

- python 설치
- pylatexenc 설치
- latex parser 설치(`:TSUpdate latex`)

### 플러그인 설정

```lua
{
  'MeanderingProgrammer/render-markdown.nvim',
  opts = {
    latex = {
      enabled = true,
      render_modes = true,
      converter = 'latex2text',
      highlight = 'RenderMarkdownMath',
      position = 'top',
      top_pad = 1,
      bottom_pad = 1,
    },
  },
},
```
