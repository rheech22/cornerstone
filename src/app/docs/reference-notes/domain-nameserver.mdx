도메인 네임서버는 도메인 이름과 IP 주소 간의 매핑을 관리하는 서버로, 사용자가 브라우저에서 주소를 입력하면 해당 도메인 이름을 IP 주소로 변환하는 역할을 한다. 브라우저는 일련의 과정을 통해 DNS 서버를 찾아 IP 주소를 받아온 후 웹 서버에 클라이언트의 요청을 보낼 수 있다.

도메인을 등록하면 기본적으로 제공되는 네임서버를 사용할 수 있다. spaceship에서 도메인을 구입했을 때에도 그랬다. 또한 도메인 등록기관의 콘솔에서 네임서버를 바꿀 수도 있는데, 가령 AWS의 Route53으로 네임서버를 바꾸고 AWS에서 [[./dns-records|도메인 레코드]]를 관리하는 것도 가능하다. 네임서버를 바꾸는 것은 [[./domain-name-register|등록기관]]을 바꾸는 것과 달리 비용이 발생하지 않는다.
