# Flow

## 문서 작성 및 동기화

1. (Desktop)터미널에서 Vimwiki 실행/(Mobile)Typewrighter 실행
2. 노트를 작성한다.
3. 변경된 내용은 iCloud에 의해 자동으로 동기화된다.

## 배포

1. commit을 하면 pre-commit hook에 의해 자동으로 iCloud의 문서가 동기화된다.
2. (수동) pnpm sync 커맨드로 스크립트를 실행하여 동기화한다.
