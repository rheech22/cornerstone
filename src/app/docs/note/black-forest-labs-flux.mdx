# Flux(FLUX.1) 모델 개요

## 정의  
- Black Forest Labs가 개발한 텍스트 → 이미지 변환을 위한 딥러닝 모델 시리즈  
- 120억(12B) 파라미터 규모의 rectified flow transformer 기반 아키텍처  
- Transformer와 diffusion 구조를 혼합한 하이브리드 아키텍처 활용

## 주요 버전 및 라이선스  
- **Schnell**: Apache-2.0 기반 완전 오픈소스 버전  
- **Dev**: 소스 제공, 비상업적 용도에 한해 사용 가능한 non-commercial 라이선스  
- **Dev**: 소스 제공, 비상업적 용도에 한해 사용 가능한 non-commercial 라이선스  
- **Pro**: 독점 API 기반 상업용 모델, 가장 높은 품질 제공 

## Flux.1-dev (일반 버전) vs Flux.1-Krea-dev 비교

### 공통점  
- 12 B 파라미터 규모의 rectified flow transformer 기반 텍스트→이미지 생성 모델  
- FLUX.1-dev 아키텍처 및 에코시스템과 완전 호환성 제공 (ComfyUI, diffusers 등)  
- 비상업적 용도 기준으로 오픈 웨이트 제공 (non-commercial 라이선스)

### 차이점  

#### 1. 미학 및 스타일 지향성  
- **Flux.1-Krea-dev**는 ‘opinionated’ 모델로, 일반적인 AI 특유의 과장된 텍스처나 인공적인 분위기(‘AI look’)를 지양하고 자연스러운 리얼리즘과 감성 중심의 표현을 목표로 설계

#### 2. 안내(distillation) 기반 훈련  
- **Flux.1-Krea-dev**는 guidance distillation 기법을 활용하여 훈련된 모델로, 보다 효율적인 생성 성능 및 스타일 반영 능력 강화됨

#### 3. 프롬프트 충실도 및 다양성  
- **Krea-dev**는 프롬프트에 대한 충실성 높은 표현력 유지하면서도, 다양한 인물 정체성, 스타일, 배경 등을 자연스럽게 생성하는 경향 보여 기존 Dev보다 다양성과 리얼리즘 향상

#### 4. 사용자 경험과 호환성  
- **Krea-dev**는 Flux.1-dev 워크플로우에 그대로 대체 가능하며 ComfyUI 및 diffusers에서도 사용 가능 (drop-in 교체 방식)

#### 5. 성능 평가지표  
- 인간 평가 기준에서 **Flux.1-Krea-dev**는 이전 오픈 모델을 능가하며, Flux 1.1 Pro와 유사 수준의 선호도 확보 (ELO 평가 기준)

#### 6. 실험 및 커뮤니티 반응  
- 일부 사용자 실험에서는 Flux.1 Dev가 프롬프트에 대해 더 엄격히 대응하였으나, **Krea-dev는 스타일적 해석을 더해 생성물에 예술적 다양성 부여**됨
