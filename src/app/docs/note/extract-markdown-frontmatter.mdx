마크다운에서 다음과 같은 frontmatter를 추출해야 함

```md
---
updated: 2025-04-19 13:28:58 +0900
created: 2025-04-19 13:28:58 +0900
title: "제목"
tags: ["태그-1", "태그-2"]
---
```

원하는 결과는 대략 다음과 같은 모양

```js
{
  updated: "2025-04-19 13:28:58 +0900",
  created: "2025-04-19 13:28:58 +0900",
  title: "제목",
  tags: ["태그-1", "태그-2"]
}
```
