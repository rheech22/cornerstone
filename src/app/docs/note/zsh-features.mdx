[Parameter Expansion](https://zsh.sourceforge.io/Doc/Release/Expansion.html#Parameter-Expansion)
- `${...}`처럼 변수의 값을 특정 규칙에 따라 바꾸거나 사용

[Expansion Modifiers](https://zsh.sourceforge.io/Doc/Release/Expansion.html#Modifiers)
- 매개변수 확장 결과에 추가적인 처리를 함

[Prompt Expansion](https://zsh.sourceforge.io/Doc/Release/Prompt-Expansion.html)
- `%x`, `%N` 등 `%`로 시작하는 특별한 코드들을 의미 있는 값(경로, 사용자 이름 등)으로 치환

**예시**

```bash  title="해당 파일의 경로" caption="프롬프트에서는 쉘의 이름(-zsh)으로 치환"
echo `${(%):-%x}`
```

```bash  title="경로에서 파일 이름은 제외" caption="디렉토리 경로만 남길 때 유용"
echo ${${(%):-%x}:h}
```
