DNS에서 레코드를 통해 IP 주소와 도메인 이름을 연결시킬 수 있다. 일반적은 레코드 종류는 다음과 같다.

## DNS 레코드의 종류

- A 레코드: 도메인 이름을 IPv4 주소에 매핑한다.
- AAAA 레코드: 도메인 이름을 IPv6 주소에 매핑한다.
- CNAME 레코드: 도메인 이름을 다른 도메인 이름에 매핑한다. 주로 서브도메인에 사용되고 IP 주소를 제공하진 않는다.
- MX 레코드: 이메일 서버를 지정하는 데 사용한다. 이메일을 이메일 서버로 전송한다.
- TXT 레코드: 도메인에 대한 추가 정보를 저장하는 데 사용한다.
