도메인을 소유하기 위해서는 도메인 등록기관을 통해 등록 과정을 거쳐야 한다. 소유권은 도메인을 등록한 개인이나 기업에게 있지만 영원한 것은 아니다. 매년 갱신 비용도 발생하고 소유권이 만료되면 다른 사람이 해당 도메인을 등록할 수 있기 때문에 사실 소유한다는 것은 틀린 표현이 아닐까..? 도메인 네임의 최대 예약 기간은 10년이고 예약 갱신만 잘한다면 무기한 갱신이 가능하기 때문에 성실하게 돈을 지불한다면 소유라고 볼 수도 있겠다.

- 도메인 등록기관은 도메인 이름의 등록과 관리에 대한 서비스를 제공한다.
- 도메인 등록기관은 ICANN(Internet Corporation for Assigned Names and Numbers)라는 국제기관의 승인을 받아 운영된다.
- 자신이 소유한 도메인의 등록기관을 이전할 수 있다. 보통 이전 비용이 발생한다.
