디아스포라(Diaspora)는 원래 고향이나 조상의 땅을 떠나 세계 각지에 흩어져 사는 민족이나 집단을 가리키는 용어

- 그리스어에서 유래, 원래는 고대 그리스인들이 지중해 각지로 이주해 정착한 현상을 설명하는 데 사용
- 역사적으로는 주로 유대인들이 예루살렘 성전 파괴 이후 세계 각지로 흩어져 살게 된 상황을 설명하는 용어로 많이 알려짐

현재는 더 넓은 의미로 사용되어, 다음과 같은 특징을 가진 집단들을 포괄

**주요 특징:**
- 원래 고향에서 떠나 다른 지역에 정착
- 고향에 대한 집단적 기억과 정체성 유지
- 흩어진 지역 간의 네트워크와 연대감
- 언젠가 고향으로 돌아갈 수 있다는 희망이나 상상

**현대적 예시:**
- 한국 디아스포라 (재외동포, 재미교포, 재일교포 등)
- 중국계 화교
- 아프리카계 미국인
- 팔레스타인 디아스포라
- 아르메니아 디아스포라
