- 과거에는 금속 활자를 배열하고 잉크를 묻힌 뒤 종이에 눌러서 인쇄본을 만들었다.
- 금속 활자는 글자와 사용 빈도 수에 따라 크고 작은 상자에 담았다.
- 각 상자는 케이스에 모았는데 대문자용 케이스를 upper-case, 소문자용 케이스를 lower-case로 불렀다.

**그리고 이러한 케이스들의 집합을 폰트라고 했다. 이 단어는 작은 금형을 만드는 주조(foundry)에서 유래했다**

- 이러한 폰트의 집합을 타입(type)이라고 불렀다.
- 글자의 굵기나 기울기 같은 변형은 페이스(face)라고 불렀다.
- 가령, 굴림체라고 한다면 이건 사실 font라기 보다는 typeface에 가깝다. 하나의 타입페이스는 여러 크기과 두깨의 폰트로 이루어져 있다.
