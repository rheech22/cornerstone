아이폰 브라우저에서 `SingleChildScrollView`에 있는 마지막 버튼이 잘 보이지 않는 현상이 있었음
게다가 왼쪽으로 다시 튕기는 현상(Bouncing Effect)로 인해 버튼을 터치하기 어려움
오른쪽에 `SizedBox`를 두어 터치할 수 있게 수정하고 버튼의 나열에서는 바운싱 효과를 주지 않는 것이 사용자 경험에 더 낫다고 판단, 안드로이드의 동작에 적용되는 ScrollPhysics인 ClampingScrollPhysics를 스크롤뷰에 적용함

```dart
// ..
        SingleChildScrollView(
          physics: ClampingScrollPhysics(),
// ..
```
