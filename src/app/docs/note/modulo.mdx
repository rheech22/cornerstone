## 어원

- 라틴어 modus는 측정, 방식, 한계라는 의미로 mode(모드), moderate(적당한), modulus(기준, 단위)의 어원
- modulo의 **어떤 기준(modulus)에 따라**를 의미
- `a modulo n`은 `a`를 `n`이라는 기준에 대해 본다라는 의미
- 핵심은 기준을 정한다라는 개념

## 수학에서의 modulo

- `a ≡ b (mod n)`는 `a`와 `b`는 `n`으로 나눈 나머지가 같음
- 이것을 `a`와 `b`는 모듈 `n`에 대한 합동 관계라고 함
- 모든 정수에 대해 `mod n`을 계산한다면, `n`개의 그룹으로 분류할 수 있음
- 이걸 **동치류(equivalence class)** 라고 함
