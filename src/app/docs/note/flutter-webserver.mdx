특정 포트로 `web-server`를 활용하여 구동
(아직 이유는 모르겠지만 `--web-browser-flag=--disable-web-security` 플래그는 잘 동작하지 않았음)

```bash
flutter run -d web-server --web-port 8080 --web-hostname 0.0.0.0
```

- `-d web-server`: 앱을 웹 서버 모드로 실행하도록 지정
- `--web-port 8080`: 웹 서버가 8080 포트에서 실행되도록 설정
- `--web-hostname 0.0.0.0`: 모든 네트워크 인터페이스에서 요청을 수신하도록 설정

IP 주소 확인 후
```bash
ifconfig | grep inet
```

모바일 브라우저에서 `http://yo.ur.ip.ip:8080`으로 접속
