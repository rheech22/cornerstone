## 정의
- 이미지 생성 모델을 위한 노드 기반 사용자 인터페이스  
- 시각적 워크플로우 편집기를 통해 모델의 파이프라인을 자유롭게 구성 가능  
- 확장과 커스터마이징에 용이한 구조  

## 주요 특징
- 극한의 작업 효율: 포토샵 불필요, ComfyUI에서 모든 작업 처리 가능
- 노코딩 자동화: 워크플로우만 구축하면 자는 동안에도 이미지 생성 가능
- 무한 확장성: 다양한 오픈소스 모델 연동 가능
- 리소스 관리: 캐싱으로 불필요한 반복작업을 줄일 수 있음
